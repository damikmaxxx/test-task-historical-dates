# Historical Dates

Тестовое задание. Блок "Исторические даты".

## Технологии
- React + TypeScript
- SCSS Modules
- GSAP (анимации)
- Swiper (слайдер)
- Webpack (сборка)

## Запуск
1. Установка зависимостей:
   `npm install`

2. Запуск в режиме разработки:
   `npm start`
   > Откроется по адресу http://localhost:3000

3. Сборка проекта:
   `npm run build`
